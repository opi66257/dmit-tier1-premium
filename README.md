## Tier1 和 Premium 到底差在哪里

一句话直答：**Tier1 走的是商业级 BGP 多线路，Premium 走的是精品直连线路（CN2 GIA / 软银 / HKBN 等），延迟更低、绕路更少，但价格也更高。**

展开说：

DMIT 的网络分层逻辑，本质上是按"回国线路质量"来划分的。

**Tier1 套餐**用的是标准商业 BGP 线路，带宽充足，适合对延迟不那么敏感的场景——比如建站、跑脚本、对外服务的业务。价格是这两类里相对亲民的一档。

**Premium 套餐**则接入了 CN2 GIA（电信精品网）、软银（SoftBank）、HKBN 等优质回程线路，专门针对中国大陆用户优化。晚高峰时段，普通 BGP 线路经常出现丢包和延迟飙升，而 Premium 的精品线路在这种时候表现明显更稳。

我把两个套餐都挂过一段时间，Tier1 在白天用基本感觉不出差距，但一到晚上 9 点到 11 点，Tier1 的延迟波动就开始明显，Premium 那台机器则稳得多。

## 哪种场景选 Tier1，哪种选 Premium

**选 Tier1 的情况：**

1. 你的主要用户或访问来源不在中国大陆
2. 跑自动化任务、爬虫、CI/CD 流水线，对延迟不敏感
3. 预算有限，想先用一台低成本机器跑通业务逻辑
4. 建站 + 静态资源托管，流量不大

**选 Premium 的情况：**

1. 你自己在国内，主要用途是日常访问、流媒体、远程办公
2. 对晚高峰延迟有明确要求（游戏、视频会议、实时传输）
3. 需要稳定的中国大陆回程，不能接受偶发性丢包
4. 已经用过 Tier1 觉得不够用，想升级

有一点值得单独说：DMIT 的 Premium 套餐在香港节点上接入了 HKBN + CN2 GIA 双回程，这个组合在香港 VPS 里算是比较少见的配置，对电信用户尤其友好。
